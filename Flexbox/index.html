Inter

Inter is a typeface carefully crafted & designed for computer screens. It is a modern sans-serif typeface that is designed for legibility across web and mobile interfaces.

Playfair Display

Playfair Display is a serif typeface with a modern twist. It is a transitional design that is inspired by the work of John Baskerville and the like. It is a great typeface for headlines and titles.

Manrope

Manrope is a modern sans-serif typeface that is designed for legibility across web and mobile interfaces. It is a great typeface for body text and headlines.
